# Security

Forgemax is designed with a defense-in-depth approach. Key features:

- No ambient capabilities in the sandbox
- Fresh runtime per call
- Pre-execution AST validation
- Credential isolation
- Worker environment clearing

See the full [SECURITY.md](https://github.com/mh0pe/forgemax/blob/main/SECURITY.md) for details and best practices.
