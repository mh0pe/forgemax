# Configuration

Configure Forgemax for your environment using the provided `forge.toml` files. See `forge.toml.example` and `forge.toml.example.production` for templates.

- **Secrets**: Store sensitive values securely.
- **Server Groups**: Isolate workflows and credentials.
- **Timeouts & Limits**: Prevent abuse and resource exhaustion.

See the [Security](./security.md) section for hardening tips.
