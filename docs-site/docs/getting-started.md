# Getting Started with Forgemax

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/postrv/forgemax.git
   ```
2. Follow the instructions in `README.md` for your platform.

## Running Forgemax

- Use the CLI to initialize and run workflows.
- See the [Configuration](./configuration.md) guide for setup details.
