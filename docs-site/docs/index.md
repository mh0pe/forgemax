# Welcome to Forgemax

Forgemax is a secure, extensible, and automated workflow platform for modern software projects. This documentation site provides everything you need to get started, configure, and extend Forgemax for your team.

- **Getting Started**: Learn how to install and run Forgemax.
- **Workflows**: Explore built-in and custom workflows, including Agentics integrations.
- **Configuration**: Secure and optimize your setup.
- **Security**: Understand the defense-in-depth model and best practices.
- **API & Extensibility**: Build your own tools and workflows.
